# ODataV4Adaptor in Syncfusion Vue Grid Component

The `ODataV4Adaptor` in the Syncfusion Vue Grid Component allows seamless integration of the Vue Grid with OData v4 services, enabling efficient data fetching and manipulation. This guide provides detailed instructions on binding data and performing CRUD (Create, Read, Update, Delete) actions using the `ODataV4Adaptor` in your Syncfusion Vue Grid Component.

## Getting Started

**1. Clone the Repository:**

Use `git clone` to fetch the repository from GitHub.

```bash
https://github.com/SyncfusionExamples/Binding-data-from-remote-service-to-vue-data-grid.git 
```

**2. Open and Build the Project:**

* Open the project in Visual Studio.
* Build the project to restore dependencies and compile it.
* Run the project

**3. Explore the Code:**

* Navigate to vue client folder(~src/app.vue)
* Debug and interact with the code as needed.
